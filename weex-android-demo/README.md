# weex-android-demo

仅供参考，请以官方API为准

### Start
这个demo基于weex sdk及官方demo，新增了多个weex页面（源码在weex-demo里面），原生与weex交互的实例。

##### 1，多WXSDKInstance实现子页面。

##### 2，从android asset目录，网络加载weex js。

##### 3，Native和Weex页面交互。

##### 4，拓展原生module，图片加载，Toast，相互通信等。

##### 5，使用Vue重写以前的demo。
